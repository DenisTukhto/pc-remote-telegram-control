# 💻 PC Remote Control Telegram Bot

Удобный Telegram-бот на Python для дистанционного мониторинга и безопасного управления компьютером под управлением ОС Windows прямо со смартфона.

## ✨ Основные возможности

* 📸 **Скриншот экрана:** Получение текущего снимка рабочего стола в режиме реального времени (`pyautogui`).
* 📊 **Телеметрия системы:** Сбор актуальных данных о нагрузке CPU, оперативной памяти и свободном месте на системном диске (C:) через `psutil`.
* 🟢 **Мониторинг NVIDIA GPU:** Безопасный последовательный опрос утилиты `nvidia-smi` (оптимизировано под RTX 5050). Выводит загрузку, температуру и энергопотребление в ваттах. Статус системы оформлен в виде аккуратного древовидного текстового списка.
* 📂 **Инлайн-Проводник файлов:** Полноценный файловый менеджер с навигацией по дискам через Inline-кнопки. Внедрен алгоритм обхода лимита Telegram в 64 байта на callback-данные с помощью словаря `path_storage`. Поддерживает скачивание файлов размером до 50 МБ.
* ⏱ **Умный таймер выключения:** Назначение завершения работы ПК кнопками-пресетами (30 мин, 1-2 часа) или отправкой произвольного текста с помощью гибкого парсера (например, `1:45`, `40 минут`, `2 часа`). Доступен сброс активного таймера.
* 🔒 **Безопасность:** Бот строго проверяет ID чата (`ALLOWED_CHAT_ID`) и полностью игнорирует запросы от любых сторонних аккаунтов.

## 🛠 Технологический стек

* **Язык:** Python 3.10+
* **Библиотеки:** `pyTelegramBotAPI` (TeleBot), `psutil`, `pyautogui`
* **Дополнительно:** CLI-утилита `nvidia-smi` (устанавливается вместе с драйверами NVIDIA)

## 🚀 Установка и запуск

### 1. Подготовка окружения
Клонируйте репозиторий и перейдите в папку проекта:

git clone https://github.com/DenisTukhto/pc-remote-telegram-control.git

cd pc-remote-telegram-control

Установите необходимые зависимости:
pip install pyTelegramBotAPI pyautogui psutil

### 2. Настройка конфигурации
> ⚠️ **ВАЖНО:** Файл `config.py` содержит конфиденциальные данные. Никогда не загружайте его на GitHub! Добавьте его в `.gitignore`.

Создайте файл `config.py` в корневой директории проекта:

# config.py
TOKEN = "ВАШ_ТОКЕН_БОТА_ОТ_BOTFATHER"

ALLOWED_CHAT_ID = 123456789  # Ваш числовой Telegram ID

START_DIR = "C:\\"          # Стартовая директория для проводника и сохранения скриншотов.

### 3. Запуск
python pc_control.py

## ⚙️ Пошаговая настройка автозапуска в фоне (Windows)

Чтобы бот автоматически запускался при старте системы и работал абсолютно скрытно (без висящего черного окна консоли), настройте **Планировщик задач Windows** строго по этой инструкции:

### Шаг 1. Открытие планировщика
1. Нажмите комбинацию клавиш `Win + R` на клавиатуре.
2. В появившемся окне «Выполнить» введите `taskschd.msc` и нажмите **Enter**.

### Шаг 2. Создание задачи
1. В правой колонке «Действия» нажмите на пункт **«Создать задачу...»** (не "простую задачу", а именно обычную).
2. На вкладке **«Общие»**:
   * **Имя:** Введите понятное название, например: `PC_Remote_Control`.
   * **Учетные записи:** По умолчанию выбрана ваша учетная запись. 
   * **Важно:** Установите галочку **«Выполнять с наивысшими правами»** (это необходимо, чтобы команды блокировки `LockWorkStation` и выключения `shutdown` срабатывали без ограничений системы).
   * **Настройка для:** В выпадающем списке внизу выберите вашу версию ОС (например, `Windows 10` или `Windows 11`).

### Шаг 3. Настройка триггера (Когда запускать)
1. Перейдите на вкладку **«Триггеры»** и нажмите кнопку **«Создать...»**.
2. В самом верхнем поле «Начать задачу:» выберите из списка **«При входе в систему»**.
3. В блоке «Параметры» выберите **«Любой пользователь»** (или конкретно ваш профиль).
4. Нажмите **ОК**.

### Шаг 4. Настройка действия (Что запускать)
1. Перейдите на вкладку **«Действия»** и нажмите кнопку **«Создать...»**.
2. В поле «Действие:» должно быть выбрано **«Запуск программы»**.
3. В поле **«Программа или сценарий»** нужно указать путь к фоновому интерпретатору Python. Нажмите «Обзор» и найдите файл **`pythonw.exe`** (буква **w** на конце критически важна, именно она скрывает черное окно консоли).
   * *Обычно он лежит тут:* `C:\Users\ИМЯ_ПОЛЬЗОВАТЕЛЯ\AppData\Local\Programs\Python\PythonXX\pythonw.exe`
4. В поле **«Добавить аргументы (необязательно)»** вставьте полный путь к файлу вашего скрипта в кавычках, например: `"C:\pc_control.py"`.
5. В поле **«Рабочая папка (необязательно)»** укажите директорию, где лежит скрипт, без кавычек: `C:\`.
6. Нажмите **ОК**.

### Шаг 5. Настройка условий и параметров (Нюансы стабильности)
1. Перейдите на вкладку **«Условия»**:
   * **Важно:** В блоке «Питание» **снимите** галочку «Запускать только при питании от электросети» (актуально для ноутбуков, иначе бот не включится без розетки).
2. Перейдите на вкладку **«Параметры»**:
   * Поставьте галочку **«При сбое перезапускать через:»** и выберите `1 мин.`. Количество попыток укажите `3`. (Если упадет интернет, бот сам поднимется).
   * В самом нижнем выпадающем списке «Если задача уже выполняется, то применить следующее правило:» выберите **«Использовать существующий экземпляр»**.
3. Нажмите **ОК** для сохранения задачи.

### Как проверить работу?
Выделите созданную задачу в списке, нажмите на неё правой кнопкой мыши и выберите **«Выполнить»**. Бот запустится в фоне. Проверьте его, отправив команду `/start` в Telegram.

## 🔒 Безопасность и архитектурные фичи

* **Белый список:** Фильтрация на уровне ID чата гарантирует, что никто кроме вас не сможет управлять компьютером.
* **Обход ограничений API:** Механизм `path_storage` хранит соответствия длинных путей и коротких ключей (например, `dir_1`, `file_5`) в оперативной памяти скрипта. Это полностью исключает ошибку `BUTTON_DATA_INVALID`, возникающую в Telegram из-за превышения лимита в 64 байта на кнопку при глубокой вложенности папок.
* **Лимиты Telegram:** Встроенный проводник автоматически скрывает файлы размером более 50 МБ, защищая бота от падения при попытке отправить слишком тяжелый файл.
